# Logos UNACH

Esta carpeta debe contener los logos oficiales de la Universidad Autónoma de Chiapas (UNACH).

## Archivos requeridos

Según la documentación de marca de UNACH, se necesitan los siguientes logos:

### 1. Logo Principal (Azul Oscuro)
- **Archivo:** `unach-logo-azul.svg` o `unach-logo-azul.png`
- **Uso:** Header principal de la aplicación
- **Color:** Azul Oscuro (RGB: 25, 45, 99)

### 2. Logo Dorado/Amarillo (Opcional)
- **Archivo:** `unach-logo-dorado.svg` o `unach-logo-dorado.png`
- **Uso:** Variante para fondos oscuros
- **Color:** Dorado (RGB: 212, 176, 18)

### 3. Logo Marrón/Dorado (Opcional)
- **Archivo:** `unach-logo-marron.svg` o `unach-logo-marron.png`
- **Uso:** Variante alternativa
- **Color:** Marrón (RGB: 115, 89, 32)

## Especificaciones

- **Formato preferido:** SVG (escalable y de alta calidad)
- **Formato alternativo:** PNG con fondo transparente
- **Tamaño recomendado:** 
  - Ancho: 200px - 300px
  - Alto: Proporcional al ancho
- **Resolución:** Mínimo 72 DPI para web, 300 DPI si se usa en documentos

## Colores oficiales UNACH

- **Azul Oscuro:** RGB(25, 45, 99) - CMYK(100, 87, 27, 13)
- **Dorado/Amarillo:** RGB(212, 176, 18) - CMYK(4, 19, 97, 18)
- **Marrón/Dorado:** RGB(115, 89, 32) - CMYK(45, 53, 100, 29)

## Nota

Si los logos no están disponibles, el sistema mostrará texto alternativo "UNACH" en su lugar.

## Instrucciones para agregar logos

1. Coloque los archivos de logo en esta carpeta (`frontend/public/logos/`)
2. Asegúrese de que los nombres de archivo coincidan con los especificados arriba
3. Los logos se cargarán automáticamente en los headers de la aplicación
4. Si un logo no se encuentra, se mostrará el texto alternativo
